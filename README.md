# Kabum Front-End
Aplicação desenvolvida em ReactJS & Redux.

----------

## Requisitos
- Node.js + NPM
- Instalar dependências (node_modules)
```bash
  npm install
```   

----------

## Endpoint Back-End API
Arquivo [consts.js][1]

----------

## Executar Aplicação
```bash
  npm start
```

## Testar Execução
